D3D11-SMAA is a header-only implementation of MLAA and SMAA for DirectX 11.

## Getting Started

Simply include `./include/agz/smaa/smaa.h` in your project.

## Building Example

Run following cmds in project directory:

```
mkdir build
cd build
cmake ..
```
